# Gemini-Robotics-ER-1.6
This project uses the Gemini API. To use the API, you need a Google AI Studio API key.

**Get an API Key**
* Sign in to Google AI Studio.
* Create a new API key.
* Copy and save the API key securely.

**Prerequisites** 

1. Install Python

2. Install Jupyter Notebook or JupyterLab

3. You can write and run the code in:

 Jupyter Notebook, JupyterLab, Visual Studio Code (VS Code)

4. Required Packages
* pip install google-genai pillow matplotlib

5. Upload an image and test.
