# Gemini-Robotics-ER-1.6
Object Detection, Zooming and Part Identification

This project demonstrates hierarchical visual understanding using Gemini Robotics-ER 1.6.

The system performs:

1. Object Detection:
    Detects the objects in a scene.
    Generates bounding box.
   
3. Target Object Localization:
   Selects a specific object of interest.
   
5. Zoomed Inspection:
   Crops and enlarges the selected object.
   
7. Part Identification:
   Detects and labels individual components of the object.
