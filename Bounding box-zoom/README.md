**Object Detection, Zooming and Part Identification**

This project demonstrates hierarchical visual understanding using Gemini Robotics-ER 1.6.

The system performs:

* Object Detection: Detects the objects in a scene. Generates bounding box.

* Target Object Localization: Selects a specific object of interest.

* Zoomed Inspection: Crops and enlarges the selected object.

* Part Identification: Detects and labels individual components of the object.
